# 🥊 UFC Fight Analytics Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard built using UFC fight data. It provides insights into fighter performance, fight outcomes, striking and grappling statistics, and key performance metrics through interactive visualizations.

## 🚀 Key Features

- Fighter Performance Analysis
- Fight Outcome Trends
- Striking & Grappling Statistics
- Win/Loss Analysis
- Interactive KPIs
- Dynamic Filters & Drill-through
